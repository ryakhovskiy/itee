itee - parent project (In-Time Enterprise Engine)
- intp -- the In-Time Platform, the server-side part of the application
- perfmon
- perfmon-ws -- the performance monitor (web-service) tool to run queries against SAP HANA and monitor the performance.
- dbloader -- the dependency of the perfmon tool that helps to execute a load of SAP HANA database
- lgenerator -- the license generator tool to generate an In-Time license
- linstaller -- the license installer tool to install the In-Time license to SAP HANA
- assembly-mtar -- the assembly project (maven) to build a mtar-archive (multi-target application)
                    for SAP HANA XSA deployment
